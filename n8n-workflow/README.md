# Weekly Dev Summary — n8n + Claude API Workflow

## Setup (5 Steps)

### 1. Import the workflow
- Open n8n → Settings → Import from File
- Select `weekly-dev-summary.json`

### 2. Configure Variables
In n8n Variables, set:

| Variable | Value |
|----------|-------|
| `GITHUB_REPO` | `https://api.github.com/repos/YOUR_ORG/YOUR_REPO` |
| `GITHUB_TOKEN` | `ghp_xxx` (fine-grained PAT with repo read access) |
| `ANTHROPIC_API_KEY` | `sk-ant-xxx` |
| `DISCORD_WEBHOOK` | Your Discord webhook URL |
| `LANGUAGE` | `EN` (English) or `FR` (French) |

### 3. Configure GitHub Token Permissions
Create a GitHub PAT with:
- `repo` (read only) for private repos
- `public_repo` for public repos

### 4. Activate the workflow
Toggle the workflow to **Active** in n8n.

### 5. Test it
Click "Test workflow" to trigger a manual run and verify output.

## What It Does

1. ⏰ **Every Friday at 5pm** (configurable) triggers automatically
2. 📊 Fetches from GitHub API: commits, closed issues, merged PRs for the week
3. 🤖 Sends data to Claude API (`claude-sonnet-4-20250514`)
4. 💬 Receives a narrative summary generated by Claude
5. 📢 Delivers via Discord webhook (or swap for email/Slack)

## Switching Delivery Channel

**Discord:** Already configured.

**Slack:**
1. Replace Discord node with Slack node
2. Use webhook URL from Slack App settings

**Email:**
1. Add Gmail/O365 node
2. Use `{{ $json.content[0].text }}` as email body

## Cron Expression
Default: `0 17 * * 5` (every Friday at 5pm UTC)

Edit the Schedule Trigger node to customize.
